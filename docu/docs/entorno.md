# Descripción del entorno y herramientas empleadas

- **Sistema base:** Alpine Linux en contenedor Docker.  
- **Servidor web:** Nginx (imagen oficial `nginx:alpine`).  
- **Gestor de contenedores:** Docker + Docker Compose.  
- **Lenguajes y tecnologías:** HTML, CSS, configuración Nginx.  
- **Herramienta de documentación:** MkDocs con tema Material.  

Se eligió Alpine por su bajo peso y eficiencia. Nginx se seleccionó por su alto rendimiento en la entrega de contenido estático.
