# Despliegue Jedi–Sith

Este proyecto tiene como objetivo desplegar y configurar un servidor web Nginx capaz de servir dos aplicaciones estáticas con dominios locales personalizados: **jedi.local:8080** y **sith.local:9090**.

Se documenta todo el proceso de instalación, configuración y pruebas de funcionamiento, aplicando principios de modularidad y escalabilidad.
