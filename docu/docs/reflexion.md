# Reflexión final

El despliegue mediante contenedores permitió configurar un entorno limpio y reproducible.  
El uso de Nginx en Alpine demostró ser ligero y eficiente para servir aplicaciones estáticas.

Como mejora futura, se podría integrar HTTPS con certificados locales y automatizar el despliegue mediante scripts o CI/CD.  
La documentación con MkDocs facilita la presentación profesional del proyecto y su mantenimiento a largo plazo.
